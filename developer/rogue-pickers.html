Rogue Pickings Website Project

This project was a static website project from my SkillCrush.com blueprint. The basis behind the project was to give us another opportunity to practice writing HTML and CSS code. We were given the PSD design and were given the task of turning it into code. The instructors did help walk us through parts of the project and gave us hints for certain parts of the code, however, I still had to problem solve for certain aspects of the project and come up with my own solutions.
